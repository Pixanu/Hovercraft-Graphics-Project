# Hovercraft Graphics

CONTROLS

(THE CONTROLS ARE FOR THE CAMERA, IN ORDER TO CHANGE THE POSITION OF IT)

W - FORWARD

S - BACKWARD

A - LEFT

D - RIGHT

LEFT CLICK - ADDS THE EXISTING LIGHTING IN THE SCENE TO THE LOCATION WHERE THE MOUSE IS LOCATED.
